Free The Lots

Tech: PHP, JS, PostgreSQL

GitHub

In collaboration with Code For KC. This project allows NLS to search the Jackson County Recorder of Deeds database to find all the Land Bank properties that a single claim holder—like Deutsche Bank—has claims against. Knowing this, would allow NLS to clear all Deutsche Bank claims against all Land Bank properties removed in a single case—rather than clearing titles for development one property at a time.

Bamazon

Tech: Node.js, JS, MySQL

GitHub

Amazon-like CLI storefront that presents users with items available for purchase. Customer mode takes purchase requests, ensures that quantities exist in the MySQL DB, calculates the cost, and asks for confirmation. Managers are able to update available stock (change existing quantity / add new items / or deplete stock) from the store's inventory.

Avoid Surge Fare

Tech: JS, Firebase, Ajax, API

GitHub

PTAE predicts when ride fares will increase, by gathering data from multiple API to analyze environmental factors (e.g., weather, local events) to estimate when Lyft will charge Prime Time pricing. The app predicts when prices will increase during a three hour span, to allow users to predict the most affordable time to use ridesharing applications.
